#!/usr/bin/env -S bunx -p https://github.com/subtleGradient/pad/archive/refs/heads/pad-oc.tar.gz pad SPEC.pad.htm SPEC.pad.htm This is the first simple PAD demo. Run this file as a program. It opens in the browser, the text is editable, and edits save back to this source file over WebSocket. No npm package, no release channel, no folders, no SSI, no new-pad generator, no ShadowRealm, no iframe, no textarea, and no browser file picker. The executable shebang and browser assets are pinned to the pad-oc GitHub branch.
