List This is one HTML file that can be read like a normal page or run as a tiny local program. The program opens this page on 127.0.0.1, makes the list blocks editable, and autosaves edits back into this same file. If You Are Nervous Do not run files from strangers. Read first, then decide. The source branch is visible at https://github.com/subtleGradient/pad/tree/pad-oc The runner is plain Bun code in pad.shebang.tsx. It serves localhost only, uses a random URL token, and writes this file when the browser sends an autosave message. The browser behavior is in pad.browser.js. It makes PAD elements editable only while the local WebSocket is healthy. The visual rules are in pad.css. There is no hidden app shell, no account, and no cloud database. How To Run It Save this file somewhere you are comfortable editing files. If needed, make it executable with: chmod +x list-example-002.pad.htm Run it with: ./list-example-002.pad.htm Your browser should open a localhost URL. The status pill should say the page is editable and autosaves to the source file. If you open the file directly in a browser instead, it stays read-only and tells you why. Try The List Click this item and type over it. Press Tab to move to the blank item at the bottom. Type into the blank item. A fresh blank item appears below it. Clear a trailing item with Backspace. Focus moves back so you can keep deleting quickly. What To Notice The empty item is not a button. It is just the next editable block waiting for text. There is always exactly one empty item at the bottom of each list. When the WebSocket disconnects, the page becomes read-only immediately and explains what happened. After a save, inspect this file again. Your edits are now part of the HTML source.
